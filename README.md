# Pulse University Festival

### Επισκόπηση
Η εργασία παρουσιάζει τον σχεδιασμό και την υλοποίηση μιας σχεσιακής βάσης δεδομένων για τη διοργάνωση του μουσικού φεστιβάλ Pulse University. Το μοντέλο καλύπτει τη δομή των φεστιβάλ, των τοποθεσιών και των σκηνών, καθώς και τη διαχείριση παραστάσεων και εμφανίσεων. Επιπλέον, υποστηρίζει δεδομένα καλλιτεχνών/συγκροτημάτων, προσωπικού και επισκεπτών με τα εισιτήριά τους. Η λειτουργικότητα συμπληρώνεται από διαδικασίες πώλησης εισιτηρίων, ουράς μεταπώλησης και συλλογής αξιολογήσεων.

### Περιεχόμενα Αποθετηρίου
- `sql/` DDL, δεδομένα, ερωτήματα και αποτελέσματα.
- `diagrams/` Διαγράμματα ER και Relational.
- `docs/report.pdf` Εργαστηριακή αναφορά.
- `code/run_queries.sh` Script εκτέλεσης ερωτημάτων (απαιτεί προσαρμογή).

### Δομή Αρχείων
```
├── code
│   └── run_queries.sh  (Script εκτέλεσης ερωτημάτων)
├── diagrams
│   ├── er.svg          (ER διάγραμμα)
│   └── relational.svg  (Relational διάγραμμα)
├── docs
│   └── report.pdf      (Εργαστηριακή αναφορά)
├── README.md
└── sql
    ├── install.sql     (DDL script)
    ├── load.sql        (Ενδεικτικά δεδομένα)
    ├── Q01.sql ...     (Απαντήσεις στα ερωτήματα 1-15)
    └── Q01_out.txt ... (Αποτελέσματα από query ερωτημάτων)
```

### Γρήγορη Χρήση
Παράδειγμα εκτέλεσης με `mysql` (προσαρμόστε χρήστη/βάση/host ανάλογα με το περιβάλλον σας):
```bash
# Δημιουργία σχήματος
mysql -u USER -p DB_NAME < sql/install.sql

# Φόρτωση ενδεικτικών δεδομένων
mysql -u USER -p DB_NAME < sql/load.sql

# Εκτέλεση ενός ερωτήματος
mysql -u USER -p DB_NAME < sql/Q01.sql
```
